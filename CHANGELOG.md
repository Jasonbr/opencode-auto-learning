# Changelog

## v1.2.0 (2025-06-16)

### Phase 4: Active Recommendation System 🤖
- Context analysis (keywords/intent/entities)
- Multi-dimensional matching
- Proactive knowledge recommendation
- Smart relevance ranking

### Phase 5: Session Summary Generator 📝
- Multi-dimensional summaries (decisions/learnings/actions)
- Multiple export formats (Markdown/JSON/Text)
- Automatic Obsidian integration
- Daily/weekly summary generation

### Phase 7: Multimodal Memory System 🖼️💻
- Image OCR with text extraction
- Code snippet indexing and search
- Automatic language detection
- Symbol extraction (functions/classes)
- Automated capture workflows

### New Commands
- `opencode-auto-learn-recommend` - Get proactive recommendations
- `opencode-summary` - Generate session summaries
- `opencode-multimodal` - Process images/code/URLs
